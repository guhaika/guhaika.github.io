如何下载ERA5再分析数据

发表于 2020-06-01 阅读次数： Valine：

1.CDS注册和登录

ERA5从2019年3月5日开始，只能使用CDS API进行下载，需要注册CDS用户，

注册链接：https://cds.climate.copernicus.eu/user/register

1.1.第一步 填写信息

提交注册后网站提示将发送邮件给你填写的邮箱

1.2.第二步 打开邮件中的第一个链接设置账号密码

设置完成后，网页提示密码已设置完成，并自动登录。

最终用户名和密码信息如下：
用户名：abcde@XXXX.com
密码：password

1.3. 第三步 在Datasets中搜索ERA5

在搜索结果中选择自己需要的数据集，比如ERA5 hourly data on single levels from 1979 to present，然后点击Download data

就可以选择自己需要的数据集中的变量、时间、经纬度范围等，格式可以选择GRIB或者NC格式。

拉到页面下方，点击Accept terms，允许当前用户下载ERA5数据。

2.下载环境说明

2.1. 安装python语言环境

安装Python是因为CDS网站提供了使用python批量下载的功能，而且有相关实例，可以方便我们编写脚本进行批量下载，如果使用其他语言可能存在接口不匹配的问题。

下载链接：https://www.python.org/getit/

执行安装并配置环境变量。

2.2. 安装pip

pip 是一个通用的 Python 包管理工具。提供了对 Python 包的查找、下载、安装、卸载的功能。使用Pip安装Python包非常方便。

下载链接：https://pypi.python.org/pypi/pip#downloads

1) 下载后解压到自定义目录，在该目录中执行如下命令进行安装
python setup.py install

2) 安装完成后执行pip命令进行检验，出现pip <command> [option]表示安装成功

2.3. 安装CDS API的客户端python库

使用pip安装客户端库，只需一条命令：pip install cdsapi

2.4. 配置CDS API密钥

安装完客户端后，还需要配置apiKey信息。

获取apikey 地址：https://cds.climate.copernicus.eu/api-how-to#install-the-cds-api-key

1) 登录后获取apikey

使用前面注册登录的id和密码进行登录，复制页面上的API Key右侧黑色背景中的内容：

url: https://cds.climate.copernicus.eu/api/v2

key: 11041:1d0374da-abcd-1024-b1d1-d6693462b4c2

2) 创建.cdsapirc文件，并将刚才复制的内容copy到该文件，然后将.cdsapirc文件放入%USERPROFILE%下，windows电脑中路径应该是C:\Users\你的用户\，本人电脑用户名为Administrator路径为C:\Users\Administrator\。

3.下载说明

下载网址：https://cds.climate.copernicus.eu/#!/search?text=ERA5&type=dataset

使用前面注册的用户名和密码登录。在步骤1.3中，选择完自己需要的数据集各项变量、时间、经纬度范围、格式等参数后，在最下面有一个Show API request按钮，点击之后会显示下载需要的api脚本代码，可以直接复制到文本保存为.py格式，运行后进行下载。

如果想要每一天下载成一个nc文件，可以使用下列代码，其中的各项参数可以自己更改。

在cmd中运行python download.py后会显示INFO Request is completed，INFO Request is running，表示正在排队，耐心等待一段时间后就会开始下载，数据文件将会下载到cmd所在的文件夹下。
